# Cookiecutter: Python + uv + ruff + pytest + ty (Python 3.13)

This repository is a **Cookiecutter template** for a modern Python project:

- **Python**: 3.13
- **Dependency manager / runner**: `uv`
- **Lint + format**: `ruff`
- **Tests**: `pytest` (dev dependency group)
- **Type checking**: `ty` (dev dependency group)

## Prerequisites

- [uv](https://docs.astral.sh/uv/)

## Usage

Generate a new project from this template:

```bash
uvx cookiecutter https://github.com/NevoleMarek/template-modern-python-package.git --checkout main
```
